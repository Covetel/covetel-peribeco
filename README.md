covetel-peribeco
================

Peribeco, LDAP Mail and Platform Admin Tool

Configuration
-------------

- Execute:

    $ cd covetel-peribeco/Peribeco

    $ ln -s ../examples/config.yml .
    $ ln -s ../example/configuracion.yml .

Run:
---

- Execute:

    $ cd covete-peribeco/Peribeco

    $ ./script/peribeco_server.pl
